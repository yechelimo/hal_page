# 个人学习页面

这是广东科学技术职业学院商学院商务数据分析与应用专业学生hal的个人学习页面。

## 项目结构

```
├── index.html          # 主页面
└── README.md           # 项目说明
```

## 功能特点

- 个人信息展示：显示姓名、学校、专业等基本信息
- 课程列表：展示多个课程卡片，包含课程名称和简要描述
- 关于我：介绍个人背景和技能
- 响应式设计：适配桌面和移动设备
- 美观的UI界面：使用Tailwind CSS构建

## 课程列表

- Python基础
- 数据分析技术
- 数据采集与处理
- 供应链数据分析
- 数据库原理与应用

## 部署到Cloudflare Pages

1. 登录Cloudflare账户
2. 进入Cloudflare Pages
3. 点击"Create a project"
4. 选择"Connect to Git"
5. 选择你的代码仓库
6. 配置构建设置：
   - 构建命令：`echo "Building static site"`
   - 构建输出目录：`/`
7. 点击"Deploy site"

## 本地预览

直接在浏览器中打开 `index.html` 文件即可预览页面。
