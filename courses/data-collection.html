数据采集与处理

高职大二第一学期

先修课程：Python基础

课程大纲

课程介绍

《数据采集与处理》是商务数据分析与应用专业的核心课程，旨在培养学生掌握数据采集的方法和数据预处理技术， 为数据分析做好准备。课程内容包括网络爬虫技术、API数据获取、数据清洗、数据转换、数据规约等核心概念和技术。

课程目标

  • 掌握数据采集的基本概念和方法
  • 熟练使用Python进行网络爬虫开发
  • 学会通过API获取数据
  • 掌握数据预处理的核心技术
  • 能够处理和转换不同格式的数据
  • 为后续的数据分析课程做好准备

1.1 数据采集的概念和重要性

数据采集的定义：从各种来源获取结构化或非结构化数据的过程

数据采集的重要性：

  • 数据分析的基础
  • 决定数据质量的关键环节
  • 获取信息的第一步

常见的数据采集方法：

  • 问卷调查
  • 日志收集
1.3 数据采集的伦理和法律问题
  • 遵守robots.txt协议
  • 注意数据隐私和安全
  • 尊重版权和知识产权
  • 合理控制采集频率

3.1 Requests库的使用
import requests

# 发送GET请求
url = 'https://example.com'
response = requests.get(url)
print(response.status_code)
print(response.text)

# 发送POST请求
data = {'key': 'value'}
response = requests.post(url, data=data)
3.2 BeautifulSoup库的使用
from bs4 import BeautifulSoup

html = requests.get(url).text
soup = BeautifulSoup(html, 'html.parser')

# 按标签查找
titles = soup.find_all('h1')

# 按class查找
items = soup.find_all(class_='product-item')

# 按ID查找
content = soup.find(id='main-content')
3.3 Scrapy框架的使用

4.2 RESTful API的使用
import requests

# 调用GitHub API
url = 'https://api.github.com/users/octocat'

4.3 API认证和授权
# 使用API密钥
headers = {'Authorization': 'Bearer YOUR_TOKEN'}
response = requests.get(url, headers=headers)

# 使用查询参数
params = {'api_key': 'YOUR_KEY'}
response = requests.get(url, params=params)
4.4 API数据格式

  • 提高数据质量

  • 节省存储空间
  • 提升分析效果
5.2 数据清洗技术
import pandas as pd

df = pd.read_csv('data.csv')

# 删除重复行
df = df.drop_duplicates()

# 填充缺失值
df['value'] = df['value'].fillna(df['value'].mean())
5.3 数据集成方法
# 合并数据
merged = pd.concat([df1, df2], axis=0)

# 按主键连接
joined = pd.merge(df1, df2, on='id')
5.4 数据变换技术
# 数据类型转换
df['date'] = pd.to_datetime(df['date'])

# 数值标准化
from sklearn.preprocessing import StandardScaler
scaler = StandardScaler()
df_scaled = scaler.fit_transform(df[['value']])
5.5 数据规约方法
# 降采样
df_sampled = df.sample(frac=0.1)

# 特征选择

df.to_csv('output.csv', index=False)
6.2 Excel文件处理

df = pd.read_excel('data.xlsx', sheet_name='Sheet1')

6.3 JSON文件处理
import json

# 读取JSON文件
with open('data.json', 'r') as f:
data = json.load(f)

# 写入JSON文件
with open('output.json', 'w') as f:
json.dump(data, f, indent=2)
6.4 XML文件处理
import xml.etree.ElementTree as ET

# 解析XML
tree = ET.parse('data.xml')
root = tree.getroot()

# 遍历元素
for child in root:
print(child.tag, child.text)
