课程介绍

《Python基础》是商务数据分析与应用专业的基础课程，旨在培养学生掌握Python编程语言的基础知识和应用技巧， 为后续的数据分析课程打下坚实的基础。课程内容包括Python的基本语法、数据类型、控制结构、函数、 面向对象编程等核心概念。

课程目标

  • 掌握Python的基本语法和编程规范
  • 熟悉Python的数据类型和数据结构
  • 掌握Python的控制结构和函数定义
  • 了解面向对象编程的基本概念
  • 能够编写简单的Python程序解决实际问题
  • 为后续的数据分析课程打下基础

1.1 Python的历史和特点

Python的历史：由Guido van Rossum于1991年创建，命名来自Monty Python喜剧团而非蛇。

Python的特点：

  • 简单易学：语法简洁清晰
  • 免费开源：可以自由使用和修改
  • 高级语言：无需关注底层细节
  • 可移植性：可在多种操作系统运行
  • 丰富的库：拥有庞大的标准库和第三方库
  • 面向对象：支持面向对象编程
1.2 Python的安装和环境配置

下载地址：https://www.python.org/downloads/

安装注意事项：

  • 勾选"Add Python to PATH"选项
  • 选择"Install Now"或自定义安装
  • 验证安装：在命令行输入 python --version

推荐的Python版本：Python 3.8或更高版本

常用开发环境：

2.1 变量和数据类型

变量命名规则：

  • 只能包含字母、数字和下划线

  • 整型(int)：如 42, -100
  • 浮点型(float)：如 3.14, 2.718
  • 字符串(str)：如 "Hello", 'World'

3.3 循环控制

break：立即退出循环

for i in range(10):
if i == 5:
break
print(i) # 输出 0-4

continue：跳过本次循环，继续下一次

for i in range(10):
if i % 2 == 0:
continue
print(i) # 输出奇数 1,3,5,7,9

4.1 列表（List）

列表特点：有序、可变、允许重复元素

# 创建列表
fruits = ["apple", "banana", "orange"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]

常用操作：

# 索引访问
print(fruits[0]) # apple
print(fruits[-1]) # orange

# 切片
print(numbers[1:4]) # [2,3,4]

# 添加元素
fruits.append("grape")
fruits.insert(1, "pear")

# 删除元素
fruits.remove("banana")
del fruits[0]

# 列表方法
numbers.sort() # 排序
numbers.reverse() # 反转
print(len(fruits)) # 长度
4.2 元组（Tuple）

元组特点：有序、不可变、允许重复元素

# 创建元组
t1 = (1, 2, 3)
t2 = ("apple", "banana")
t3 = (1,)

元组操作：

# 索引和切片
print(t1[0]) # 1

# 元组解包
a, b, c = t1
print(a, b, c)
4.3 字典（Dictionary）

字典特点：键值对、无序、键唯一、可变

# 创建字典
student = {
"name": "张三",
"age": 20,
"major": "计算机"
}

字典操作：

# 访问值
print(student["name"])
print(student.get("age", 18))

# 添加/修改
student["grade"] = 3
student["age"] = 21

# 删除
del student["major"]

# 遍历
for key, value in student.items():
print(key, value)
4.4 集合（Set）

集合特点：无序、不重复、可变

# 创建集合
s1 = {1, 2, 3, 4}
s2 = set([1, 2, 2, 3, 3, 3])

集合操作：

# 添加/删除
s1.add(5)
s1.remove(1)

# 集合运算
print(s1 & s2) # 交集
print(s1 | s2) # 并集
print(s1 - s2) # 差集
4.5 数据结构的应用

列表推导式：

# 生成1-10的平方
squares = [x**2 for x in range(1, 11)]

# 过滤偶数
evens = [x for x in range(1, 11) if x % 2 == 0]

5.1 函数的定义和调用

函数定义语法：

def function_name(parameters):
"""函数文档字符串"""
# 函数体
return value

例子：

def greet(name):
"""向指定的人打招呼"""
print(f"Hello, {name}!")

greet("张三") # 调用函数
5.2 函数参数

位置参数：按顺序传递

关键字参数：按参数名传递

默认参数：设置默认值

可变参数：*args和**kwargs

def add(a, b=10):
return a + b

print(add(5)) # 15
print(add(3, 4)) # 7
print(add(b=5, a=10)) # 15

def sum_all(*args):
return sum(args)
print(sum_all(1, 2, 3, 4)) # 10
5.3 函数返回值
def calculate(a, b):
return a + b, a - b, a * b

sum_val, diff_val, product_val = calculate(10, 5)
print(sum_val, diff_val, product_val) # 15, 5, 50
5.4 递归函数

阶乘例子：

def factorial(n):
if n == 0 or n == 1:
return 1
return n * factorial(n - 1)

print(factorial(5)) # 120
5.5 lambda表达式

匿名函数语法：

add = lambda x, y: x + y
print(add(3, 5)) # 8

结合map和filter使用：

numbers = [1, 2, 3, 4, 5]
squares = list(map(lambda x: x**2, numbers))
print(squares) # [1,4,9,16,25]

evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens) # [2,4]

self.brand = brand

def __del__(self):
print(f"{self.color} {self.brand} 销毁了")

6.3 继承和多态

继承：

class Animal:
def __init__(self, name):
self.name = name

def speak(self):
pass

class Dog(Animal):
def speak(self):
return f"{self.name} 汪汪叫"

class Cat(Animal):
def speak(self):
return f"{self.name} 喵喵叫"

多态：

def make_speak(animal):
print(animal.speak())

dog = Dog("旺财")
cat = Cat("咪咪")

make_speak(dog)
make_speak(cat)
6.4 封装和抽象

私有属性和方法：使用双下划线 __

class BankAccount:
def __init__(self, balance):
self.__balance = balance # 私有属性

def deposit(self, amount):
if amount > 0:
self.__balance += amount

def get_balance(self):
return self.__balance

account = BankAccount(1000)
account.deposit(500)
print(account.get_balance()) # 1500
6.5 面向对象编程实战

学生管理系统例子：

class Student:
def __init__(self, name, student_id, grades=None):
self.name = name
self.student_id = student_id
self.grades = grades or []

def add_grade(self, grade):
self.grades.append(grade)

def get_average(self):
return sum(self.grades) / len(self.grades) if self.grades else 0

class StudentManager:
def __init__(self):
self.students = []

def add_student(self, student):
self.students.append(student)

def find_student(self, student_id):
for s in self.students:
if s.student_id == student_id:
return s
return None

def is_prime(n):
